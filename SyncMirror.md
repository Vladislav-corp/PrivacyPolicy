## Privacy Policy for SyncMirror

**Effective date:** April 2026

SyncMirror is a macOS application designed to synchronize and back up folders selected by the user.

### Data Collection

SyncMirror does **not collect, store, process, or share any personal data**.

The app:

* does not require user accounts
* does not use analytics services
* does not track usage
* does not use third-party SDKs
* does not transmit any data to external servers

All functionality runs locally on your device.

### File Access

SyncMirror accesses only the folders you explicitly select through the system file picker.
No file contents or metadata are sent outside your device.

### Changes

If this policy changes in the future, updates will be posted on this page.
